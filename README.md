Basic Listing Website
=====================

Alicia Sedarski
---------------

Built with:
  * Symfony PHP framework
  * MVC framework format
  * Twitter Bootstrap 3
  * NPM
  * Gulp
  * Bower
  * jQuery
  * Composer
  * PHP SendMail

To run locally:
  * run `composer update`
  * run `php bin/console server:run`
  * Open <http://localhost:8000/>
